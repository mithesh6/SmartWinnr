# SmartWinnr Frontend 🎨

Angular-based dashboard for real-time analytics and management.

## 🚀 Version Requirements
- **Angular**: 21.2.x
- **Node.js**: v18.x / v20.x
- **npm**: v10.x+

## 💻 Tech Stack
- **Framework**: Angular 21 (Standalone Components)
- **State**: RxJS Observables & Signals
- **Charts**: Chart.js 4.5.x
- **Reports**: jsPDF & File-Saver
- **Real-time**: Socket.io-client

## 🛠️ Local Setup
1.  `npm install`
2.  `npm start`
3.  Open `http://localhost:4200`

## 📦 Features
- **Glassmorphism Design System**
- **SearchService**: Global search synchronization.
- **Reporting**: Excel/PDF exports.
- **Dark Mode**: Persistent theme settings.
